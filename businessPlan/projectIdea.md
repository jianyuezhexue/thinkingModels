# 思维模型平台项目构想

## 项目概述
通过结构化思维和AI能力，帮助用户快速精准地完成对复杂课题的思考，或将想法落地。

## 核心功能
1. 思维模型市场
2. 我的课题
3. 思维碰撞
4. 找人聊聊
5. 找自己
6. 思维练习

## 具体功能描述
思维模型市场：提供经过精打细磨的思维模型模板，用户可上传自己的模型并定价。
我的课题：用户记录自己的思考课题，可选用多个思维模型辅助分析，最终导向具体行动或解决问题。
思维碰撞：提供公共讨论空间，用户分享想法，大家共同讨论，初期采用论坛式讨论，后续迭代升级。
找人聊聊：用户发布话题和地点，选择咖啡费用承担方式（我来请或AA制），实现线下见面讨论，设计用户评分机制保障交流质量。
找自己：用户介绍自己的思想特点、人格特质，展示当前思考的课题，信息可收录到"找人聊聊"板块作为线下活动参考。
思维练习：提供多种思维模型，用户可自由选择，进行训练，提升思维能力。


## 商业模式
- 免费用户：每周10次对话名额
- 年度付费用户：89元/年
- 终生用户：399元
- 企业用户：20元/席位/月

## 技术栈
- 后端：GoLang, Gin
- 前端：Vue, Element UI Plus
- 存储：MySQL, Redis
