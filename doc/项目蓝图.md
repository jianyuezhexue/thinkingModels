# 思维模型项目蓝图

## 项目愿景

**让每个人都能站在巨人的肩膀上思考，用最优秀的思维模型提升认知、解决生活与工作中的问题、持续创造价值。**

---

## 核心概述

思维模型项目旨在构建一个**思维模型的汇聚与应用平台**，实现以下核心目标：

1. **收集顶尖思维模型** - 汇聚世界各领域最优秀的思维模型，形成可检索、可学习的知识库
2. **赋能用户创造** - 用户可将自己的思维模型，思考方式、方法论、SOP、经验总结抽象整理为可复用的思维模型
3. **市场化流通** - 通过思维模型市场，让优质模型得以传播，供自己和他人使用；**创作者可对模型定价，用户采纳付费模型后支付相应费用，实现知识变现**
4. **深度思考实践** - 结合思维模型 + AI 对话，或邀请他人协作讨论，实现系统性、结构化的深度思考
5. **知行合一闭环** - 思考导向行动，通过具体实践内化思维模型，持续学习成长、迭代升级
6. **构建智性社群** - 思考的最终目的是导向生活；通过「找人聊聊」板块促进用户间轻松交流，「付费咨询」板块连接领域专家，打造一个以深度思考为纽带的智性社群

---

## 主程用例

### 用例一：从困惑到行动的深度思考之旅

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                           用户核心操作流程                                      │
└────────────────────────────────────────────────────────────────────────────────┘

    ①                  ②                  ③                  ④                  ⑤
┌─────────┐        ┌─────────┐        ┌─────────┐        ┌─────────┐        ┌─────────┐
│ 准备工具 │   ->   │ 开启课题 │   ->   │ 深度思考 │   ->   │ 产出行动 │   ->   │ 跟进闭环 │
└─────────┘        └─────────┘        └─────────┘        └─────────┘        └─────────┘
```

**① 准备工具：获取思维模型**
- 浏览思维模型市场，发现适合的思维框架
- 购买/采纳心仪的付费或免费思维模型
- 或自行创建、抽象整理个人思维模型

**② 开启课题：定义思考主题**
- 新建一个思考课题（可以是困惑、决策、规划等）
- 为课题加载一个或多个相关思维模型
- 讲述自己对该课题的初步思考、背景信息或核心困惑

**③ 深度思考：多维探索与碰撞**
- 与 AI 进行结构化对话，借助思维模型深入分析
- 邀请他人（朋友、专家、社群成员）参与课题讨论
- 多角度碰撞思想，系统性梳理问题与可能性

**④ 产出行动：从思考到计划**
- 将讨论成果转化为具体、可执行的行动计划
- 拆解任务，明确下一步行动与时间节点
- 确认行动方案，准备落地执行

**⑤ 跟进闭环：内化与成长**
- 跟进行动执行进度，记录实践反馈
- 复盘思考过程与行动效果
- 内化思维模型，完成对课题的完整思考闭环

### 用例二：成为思维模型专家，实现知识变现

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                         创作者知识变现流程                                       │
└────────────────────────────────────────────────────────────────────────────────┘

    ①                  ②                  ③                  ④                  ⑤
┌─────────┐        ┌─────────┐        ┌─────────┐        ┌─────────┐        ┌─────────┐
│ 沉淀经验 │   ->   │ 创建模型 │   ->   │ 发布市场 │   ->   │ 持续运营 │   ->   │ 睡后收入 │
└─────────┘        └─────────┘        └─────────┘        └─────────┘        └─────────┘
```

**① 沉淀经验：提炼智慧结晶**
- 回顾自身在某一领域的经验、方法论、成功模式
- 识别可复用的思考方式、决策框架或 SOP 流程
- 将隐性知识显性化，梳理核心要素与应用场景

**② 创建模型：结构化表达**
- 使用平台工具创建思维模型
- 定义模型名称、适用场景、核心步骤与引导问题
- 配置 AI 辅助对话模板，提升模型使用体验
- 打磨细节，确保他人也能顺畅应用

**③ 发布市场：上架与定价**
- 选择免费分享或付费发布
- 为付费模型设定合理价格
- 撰写模型介绍与使用说明，吸引目标用户
- 提交审核，正式上架思维模型市场

**④ 持续运营：打造个人品牌**
- 收集用户反馈，持续优化模型内容
- 回应用户评论与问题，建立信任
- 通过「找人聊聊」「付费咨询」接触更多用户
- 逐步建立个人专家品牌与影响力

**⑤ 睡后收入：知识持续变现**
- 每当有新用户采纳付费模型，自动获得收益
- 模型被更多人使用，形成口碑传播
- 构建思维模型矩阵，多模型组合增加收入来源
- 实现知识资产化，智慧持续创造价值

### 模块一：我的思维模型

| 功能 | 描述 |
|------|------|
| **新建模型** | 创建个人思维模型，通过三步流程完成创建（详见下方操作流程） |
| **编辑模型** | 修改已创建模型的内容、定价等设置 |
| **发布模型** | 将模型提交审核，审核通过后发布到市场 |

#### 新建模型 - 详细操作流程

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                         创建思维模型 - 分步表单流程                             │
└────────────────────────────────────────────────────────────────────────────────┘

    STEP 1                    STEP 2                    STEP 3
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   📋 基本信息    │  ->  │   📝 内容编辑    │  ->  │   🚀 发布设置    │
│                 │      │                 │      │                 │
│ 模型名称、分类、 │      │ 使用步骤和案例  │      │ 定价和发布选项  │
│    简介         │      │                 │      │                 │
└─────────────────┘      └─────────────────┘      └─────────────────┘
```

**STEP 1：基本信息**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 1.1 | 选择/上传封面 | 点击选择预设图片（24张高质量免费图片可选）或上传自定义图片；支持更换和移除 | 否 |
| 1.2 | 输入模型名称 | 为思维模型命名，长度 2-50 字符，如"SWOT 分析模型" | ✅ |
| 1.3 | 输入模型描述 | 简要描述模型的用途、适用场景和价值，长度 10-500 字符 | ✅ |
| 1.4 | 选择模型分类 | 从 8 个预设分类中选择：商业管理、战略规划、创新思维、分析工具、决策方法、创意构思、心理学、沟通表达 | ✅ |
| 1.5 | 添加标签 | 手动添加或从推荐标签中选择，最多 5 个标签；推荐标签包括：战略、分析、思维、创新、管理、决策、效率、逻辑、沟通、规划 | 否 |
| 1.6 | 点击「下一步」 | 系统校验必填项，通过后进入 STEP 2 | - |

**STEP 2：内容编辑**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 2.1 | 编写模型概述 | 详细介绍模型的背景、理论基础、核心原理和适用场景 | ✅ |
| 2.2 | 添加使用步骤 | 按顺序列出使用该模型的详细步骤，每个步骤包含：步骤标题 + 详细描述；至少需要一个完整步骤；可点击「+ 添加步骤」增加更多步骤 | ✅ |
| 2.3 | 删除步骤 | 点击步骤右侧的删除按钮移除该步骤（至少保留一个） | - |
| 2.4 | 添加实践案例 | 提供真实或假设的应用案例，每个案例包含：案例标题 + 详细内容；可点击「+ 添加案例」增加更多案例 | 否 |
| 2.5 | 删除案例 | 点击案例右侧的删除按钮移除该案例（至少保留一个） | - |
| 2.6 | 点击「下一步」 | 系统校验概述和步骤完整性，通过后进入 STEP 3 | - |

**STEP 3：发布设置**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 3.1 | 选择定价模式 | 选择「🆓 免费」或「💎 付费」；免费可获得更多曝光，付费可实现知识变现 | ✅ |
| 3.2 | 设置价格 | 若选择付费：手动输入金额（¥1-999）或从快捷价格中选择（¥9/19/29/49/99）；系统显示平台服务费（20%）和预计收入 | 付费时必填 |
| 3.3 | 阅读发布须知 | 了解审核流程（24小时内）、内容原创要求、发布后影响、收入结算规则（T+7） | - |
| 3.4 | 点击「提交审核」 | 弹出确认对话框，确认后模型进入审核流程 | - |

**辅助操作**

| 操作 | 说明 |
|------|------|
| **保存草稿** | 任意步骤可点击「保存草稿」，保存当前进度后返回我的模型列表 |
| **上一步** | 返回上一个步骤，已填写内容保留 |
| **取消** | 在 STEP 1 可点击取消，放弃创建并返回上一页 |
| **查看预览** | 右侧边栏实时显示模型预览卡片，包含封面、名称、描述、标签、分类、价格 |
| **查看创作指南** | 右侧边栏提供创作指南和小贴士，帮助用户创建优质模型 |
| **完成度指示** | 顶部显示当前完成进度百分比 |

```
┌────────────────────────────────────────────────────────────────────────────────┐
```

```
└────────────────────────────────────────────────────────────────────────────────┘

    STEP 1                    STEP 2                    STEP 3
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   📋 基本信息    │  ->  │   📝 详细描述    │  ->  │   🚀 确认创建    │
│                 │      │                 │      │                 │
│ 课题标题与背景  │      │ 目标与约束条件  │      │ 检查并提交      │
└─────────────────┘      └─────────────────┘      └─────────────────┘
```

**STEP 1：基本信息**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 1.1 | 输入课题标题 | 用一句话概括要思考的问题，长度 2-100 字符，例如"如何提高团队的创新能力？" | ✅ |
| 1.2 | 输入背景说明 | 描述课题产生的原因和背景，例如"团队目前的创新速度跟不上市场需求..." | 否 |
| 1.3 | 添加标签 | 手动输入或从推荐标签中选择，最多5个；推荐标签：商业决策、产品规划、团队管理、职业发展、创新思维、问题解决、战略分析、个人成长 | 否 |

| 1.5 | 点击「下一步」 | 系统校验必填项（标题），通过后进入 STEP 2 | - |

**STEP 2：详细描述**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 2.1 | 设定目标 | 描述期望达成的具体目标，例如"在3个月内将新想法的落地速度提升50%" | 否 |
| 2.2 | 填写约束条件 | 列出需要考虑的限制因素，例如"预算有限，需要利用现有资源" | 否 |
| 2.3 | 点击「下一步」 | 进入 STEP 3 确认页面 | - |

**STEP 3：确认创建**

| 序号 | 操作项 | 操作说明 | 必填 |
|------|--------|----------|------|
| 3.1 | 预览课题信息 | 查看整合后的课题预览：标题、标签、完整描述（包含背景、目标、约束条件） | - |
| 3.2 | 查看创建后提示 | 了解创建后可进行的操作：选用思维模型、多次分析对比、保存分析结果 | - |
| 3.3 | 点击「创建课题」 | 提交创建，成功后自动跳转到课题详情页 | - |

| **完成度指示** | 顶部显示当前完成进度百分比（根据填写项计算）|

**辅助功能（右侧边栏）**

| 功能 | 说明 |
